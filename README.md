# Crypto Market Sentiment Analysis

## Project Overview
This project analyzes the relationship between cryptocurrency market sentiment and trading performance using historical trading data and the Fear & Greed Index.

## Objectives
- Explore the impact of market sentiment on trading outcomes
- Identify patterns in trader performance
- Visualize sentiment-driven trends
- Generate actionable insights from data

## Dataset
1. historical_data.csv
2. fear_greed_index.csv

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Git & GitHub

## Key Analysis Performed
- Data Cleaning
- Data Merging
- Exploratory Data Analysis (EDA)
- Sentiment Classification Analysis
- Visualization and Insights

## Results
- Compared trader performance across Fear, Greed, Extreme Fear and Extreme Greed market conditions.
- Identified sentiment categories associated with higher average profitability.
